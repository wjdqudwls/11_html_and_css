CSS 기본 선택자

선택자(selector)란? 특정한 HTML 요소를 선택할때 사용하는 기호

태그 선택자

    HTML 문서내에 지정된 태그를 가진 갖는 모든 요소를 선택하는 선택자

p { color : red; } 작성하기

ID 선택자 (#)

    - HTML 문서 내에 해당하는 아이디 속성을 가진 태그만 선택할 때 사용하는 선택자
    - HTML 문서 내에 중복되는 아이디를 가지는 요소는 존재해선 안된다!
  
    [작성법]
    #id 속성값 { css 코드 }

Class 선택자(.)

    HTML 문서 내에 여러 개의 요소를 동일한 클래스명을 부여하고 선택할 때 사용한다.(여러 요소 선택 시 주로 사용)

    [작성법]
    .class 속성명 { css 코드; }
